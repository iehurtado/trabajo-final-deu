## Project Outline

This project uses yarn package manager to manage two workspaces:

- `backend`: NestJS API with OpenAPI doc support.
- `frontend`: Angular application that serves as frontend.

## TypeScript Best Practices

- Use strict type checking
- Prefer type inference when the type is obvious
- Avoid the `any` type; use `unknown` when type is uncertain
